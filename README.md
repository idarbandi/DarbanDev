## 🎨 Customization

### Edit Content
- Open `index.html` and update the text content
- Use `data-en` for English and `data-fa` for Persian text

### Change Colors
- Edit CSS variables in `styles.css`:
```css
:root {
  --primary: #0a66c2;        /* Main color */
  --secondary: #67b26f;      /* Accent color */
  --background: #ffffff;      /* Background */
  /* ... more variables */
}
```

### Add New Sections
- Add new section in HTML
- Add styles in CSS
- Use the same bilingual structure with `data-en` and `data-fa`

## 🔧 Installation

```bash

cd DarbanDev

# Open in browser
open index.html  # macOS

```

## 🚀 Deployment

### GitHub Pages (Free)

1. Go to repository settings
2. Navigate to **Pages**
3. Select `main` branch as source
4. Save and wait for deployment

Your site will be live at: `https://idarbandi.github.io/DarbanDev`

### Custom Domain
1. Add CNAME file with your domain
2. Configure DNS settings with your provider
3. Enable HTTPS in GitHub Pages settings
